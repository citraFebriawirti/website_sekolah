- Login (sesuai Username/Password)
- Register
- Beranda
- Tentang Kami
- Kegiatan
- Berita
- Kontak

## Instalasi
- Download semua file
- Taruh di htdocs XAMPP
- Buat database dengan nama "db_sekolah" di MySQL Server
- Import SQL File dari folder database (gunakan yang db_sekolah)
